## 一 大 O 表示法

| 符号 | 名称 |
| --- | --- |
| O(1) | 常数的 |
| O(log(n)) | 对数的 |
| O((log(n))c) | 对数多项式的 |
| O(n) | 线性的 |
| O(n^2) | 二次的 |
| O(n^c) | 多项式的 |
| O(c^n) | 指数的 |
