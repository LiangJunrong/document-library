* **题目内容**：

```
学校在拍年度纪念照时，
一般要求学生按照 非递减 的高度顺序排列。

请你返回能让所有学生以 非递减 高度排列的最小必要移动人数。

示例：

输入：heights = [1,1,4,2,1,3]
输出：3

提示：

1 <= heights.length <= 100
1 <= heights[i] <= 100

来源：力扣（LeetCode）
链接：https://leetcode-cn.com/problems/height-checker
```
