下面是动态规划求解：

> 动态规划

```js
const respace = (dictionary, sentence) => {
  const dp = [ 0 ];
  for (let i = 1; i <= sentence.length; i++) {
    dp[ i ] = dp[i - 1] + 1;
    for (let item of dictionary) {
      if (sentence.substring(i - item.length, i) === item) {
        dp[i] = Math.min(dp[i], dp[i - item.length]);
      }
    }
  }
  return dp[dp.length - 1];
};
```
